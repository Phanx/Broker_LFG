--  Description  -------------------------------------------------------

	Asks for confirmation when you attempt to leave a party or raid
	group while you are in a PvE instance.

--  Usage  -------------------------------------------------------------

	There are no options. It "just works".

--  Localization  ------------------------------------------------------

	Compatible with all locales.

	Translated into English. If you'd like to provide a translation for
	the confirmation message, post a comment on the download page.
